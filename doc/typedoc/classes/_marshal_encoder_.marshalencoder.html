Class MarshalEncoder

Provides Marshalling Services for ThreadBox. Allows constructors to be registered as marshalled, and provides logic to encode and decode objects crossing thread boundaries. MarshalEncoder is leveraged by the Worker and Channel protocols to facilitate complex message passing.
